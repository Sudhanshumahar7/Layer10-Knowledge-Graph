# Layer10 Frontend (React)

A professional react-based dashboard for visualizing and exploring organizational knowledge.

## Setup & Running

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Start the development server**:
   ```bash
   npm start
   ```

## Features

- **Knowledge Graph**: Physics-based interactive visualization.
- **Analytics**: Real-time charts of graph statistics.
- **Node Explorer**: Browse evidence, quotes, and source URLs for every relationship.
